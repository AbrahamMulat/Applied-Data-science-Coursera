### Lists and Tuples
1. Consider the following tuple:

```python 
say_what=('say',' what', 'you', 'will') 
```
what is the result of the following ``` say_what[-1] ```


- [ ] 'you'
- [ ] 'say'
- [ ] ' what'
- [x] 'will'

2. Consider the following tuple ``` A=(1,2,3,4,5) ```. What is the result of the following: ``` A[1:4] ```:

- [ ] (3, 4,5)
- [ ] (2, 3, 4,5)
- [x] (2, 3, 4)

- [ ] 6
